## Binary Release Requirements

Each binary release must include or link to:

- Corresponding source code for the exact binary.
- Inkscape license material.
- Ink/Stitch license material.
- Third-party dependency licenses.
- A changelog that describes modifications.
- Build instructions sufficient to reproduce the modified source build.
